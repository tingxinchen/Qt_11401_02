# Qt_11401_02

## Qt 簡易通訊軟體

這是一個使用 Qt5 開發的簡易通訊軟體，實現了以下功能：
- ✅ 一對一通訊
- ✅ 好友名單管理

### 程式介面預覽

![主視窗 - 好友名單](https://github.com/user-attachments/assets/b19c1ab1-6961-4a01-bfd8-c44a8e7fa10c)

*主視窗顯示好友名單、新增/刪除好友按鈕以及伺服器設定功能*

### 功能特色

#### 好友名單
- 新增好友：可以將好友加入到名單中
- 刪除好友：可以從名單中移除好友
- 好友資料持久化：好友名單會自動儲存，下次啟動時會自動載入

#### 一對一通訊
- TCP/IP 通訊：使用 TCP 協議進行穩定的通訊
- 伺服器模式：可以啟動伺服器等待其他人連接
- 客戶端模式：可以連接到其他人的伺服器
- 雙向即時通訊：支援即時發送和接收訊息
- 時間戳記：每則訊息都會顯示發送時間
- 連線狀態顯示：清楚顯示當前連線狀態

### 系統需求

- Qt 5.x 或更高版本
- C++11 或更高版本的編譯器
- 支援的作業系統：Linux, Windows, macOS

### 編譯方式

#### Linux / macOS

```bash
# 1. 安裝 Qt5 開發工具
# Ubuntu/Debian:
sudo apt-get install qt5-qmake qtbase5-dev qtbase5-dev-tools build-essential

# 2. 編譯專案
qmake ChatApp.pro
make

# 3. 執行程式
./ChatApp
```

#### Windows

```bash
# 1. 安裝 Qt5 (從 https://www.qt.io/download)
# 2. 使用 Qt Creator 開啟 ChatApp.pro
# 3. 點擊「建置」和「執行」
```

### 使用說明

#### 設定好友名單
1. 啟動程式後，在主視窗可以看到「好友名單」
2. 點擊「新增好友」按鈕，輸入好友名稱
3. 雙擊好友名稱即可開啟聊天視窗

#### 進行一對一通訊

**方式一：作為伺服器端**
1. 在主視窗點擊「啟動伺服器」
2. 輸入連接埠號（例如：8888）
3. 等待對方連接
4. 當對方發送訊息時，會自動開啟聊天視窗

**方式二：作為客戶端**
1. 雙擊好友名稱開啟聊天視窗
2. 點擊「連接」按鈕
3. 輸入對方的 IP 位址和連接埠
4. 連接成功後即可開始聊天

#### 發送訊息
1. 在聊天視窗的訊息輸入框中輸入文字
2. 點擊「發送」按鈕或按 Enter 鍵
3. 訊息會顯示在聊天記錄中

### 架構說明

```
ChatApp/
├── ChatApp.pro         # Qt 專案檔
├── main.cpp            # 程式入口點
├── mainwindow.h/cpp/ui # 主視窗（好友名單）
└── chatwindow.h/cpp/ui # 聊天視窗（一對一通訊）
```

### 技術細節

- **UI 框架**：Qt Widgets
- **網路通訊**：Qt Network (QTcpServer, QTcpSocket)
- **資料儲存**：文字檔案（friends.txt）
- **訊息格式**：UTF-8 編碼

### 注意事項

1. 防火牆設定：確保防火牆允許程式使用指定的連接埠
2. 網路環境：在同一區域網路下使用效果最佳
3. IP 位址：本機測試可使用 127.0.0.1（localhost）
